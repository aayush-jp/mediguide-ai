MediGuide-AI

AI-powered healthcare assistant that combines symptom analysis, disease prediction, medical report OCR, multilingual medical chatbot support, and intelligent risk assessment using AI agents, MCP servers, and machine learning models. MediGuide-AI helps users understand possible health conditions, analyze medical reports, and receive safe next-step recommendations while escalating serious cases to medical professionals when necessary.

Features
AI symptom checker
Disease prediction using ML
Medical chatbot with follow-up questioning
OCR analysis for medical reports and prescriptions
Voice input support
Multi-language support
Intelligent risk triage system
AI agent + MCP server architecture
Health trend tracking and report summaries
Safe doctor escalation system for serious conditions
Tech Stack
React / Next.js
FastAPI + Python
Transformers / BioBERT / XGBoost
OCR (Tesseract / EasyOCR)
MongoDB / PostgreSQL
MCP tools & AI agent workflows

Built as an AI-assisted healthcare support platform — not a replacement for licensed medical professionals.
